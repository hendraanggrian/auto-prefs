Saver

interface Saver

Interface used for applying changes made in Editor object. When using ReadablePrefs.bind to inject preferences' values to JVM fields, Saver is returned back to save changes made in JVM fields.

Functions

save

Commit your preferences changes back from Editor to the ReadablePrefs object it is editing. This atomically performs the requested modifications, replacing whatever is currently in the ReadablePrefs.
