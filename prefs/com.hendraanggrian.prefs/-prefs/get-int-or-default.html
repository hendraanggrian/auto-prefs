getIntOrDefault

open fun getIntOrDefault(key: String, defaultValue: Int): Int

Retrieve an Int value from the preferences, providing default value as a fallback.

Parameters

key - name of the preference to retrieve.

defaultValue - value to return if this preference does not exist.

Exceptions

UnsupportedOperationException - if Int is not a supported type of this preferences.

Return
preference value if it exists, or default value.
