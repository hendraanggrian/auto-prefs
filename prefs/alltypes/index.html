All Types

com.hendraanggrian.prefs.EditablePrefs

Subinterface of Prefs that, in contrast to SimplePrefs, requires an instance of Prefs.Editor to modify preferences.

com.hendraanggrian.prefs.Prefs

Interface for accessing and modifying preference data on any target platform (JVM, Android, etc.).

com.hendraanggrian.prefs.internal.PrefsBinding

Parent class of any class generated by local-compiler (should they are being used in the project). It behaves similarly as Prefs with simpler API for generated sources to use.

com.hendraanggrian.prefs.SimplePrefs

Subinterface of Prefs that, in contrast to EditablePrefs, can directly modify preferences since it also inherits Prefs.Editor.
